Cómo usar este plan

  1. 1
    Elige el siguiente item según el cuadro de prioridades (empieza por los P0, de arriba hacia abajo).
  2. 2
    Abre su prompt.md y pega el prompt completo en Claude Code dentro del repo. La auditoría generará REPORTE.md con hallazgos y BUENAS_PRACTICAS.md con las reglas que Claude Code seguirá desde entonces.
  3. 3
    Abre despues-de-la-auditoria.html del mismo item y aplica los fixes en orden: primero 🔴 crítico, luego 🟠 importante, al final 🟢 opcional.
  4. 4
    Marca el item como hecho (columna Estado del reporte) y pasa al siguiente. Re-corre la auditoría cada 2–3 meses o tras cambios grandes en esa área.

Niveles de prioridad

P0 · Crítico

Hazlo esta semana

Riesgo directo de pérdida de dinero, datos de clientes o caída del negocio.

P1 · Importante

Hazlo este mes

Impacto alto en ventas, confianza o mantenibilidad; no es incendio, pero no lo dejes enfriar.

P2 · Mejora

Cuando lo anterior esté hecho

Optimizaciones que suman; hazlas de forma continua después de P0 y P1.

🔐 Seguridad

Protege el sitio, el panel admin y los datos de tus clientes contra accesos no autorizados y abuso.

  • Autenticación y autorización del panel admin P0 · Crítico

  • Validación de inputs y subida de archivos P0 · Crítico
  • CSP y headers de seguridad P1 · Importante
  • Rate limiting y anti-abuso P1 · Importante
  • Secretos y dependencias vulnerables P1 · Importante

💳 Pagos y dinero

Que cada peso que ve el cliente sea el que se cobra, que ningún pedido pagado se pierda y que nadie pueda manipular precios.

  • Integridad de precios end-to-end P0 · Crítico
  • Webhooks de pago (PayPal / Mercado Pago / legacy) P0 · Crítico
  • Conciliación de pedidos y reembolsos P1 · Importante
  • Códigos de descuento y promociones P2 · Mejora

⚡ Rendimiento

Velocidad de carga y fluidez, sobre todo en móvil: cada segundo extra en el checkout cuesta conversiones.

  • Core Web Vitals (Lighthouse) P1 · Importante
  • Optimización de imágenes P1 · Importante
  • Bundle JS y peso del checkout P1 · Importante
  • Caching de API y datos P2 · Mejora

🛒 Conversión y UX de venta

Que el máximo de visitantes termine pagando: menos fricción, más confianza y una experiencia móvil impecable.
