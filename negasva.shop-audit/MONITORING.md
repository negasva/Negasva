# Monitoring Plan - negasva.shop

## Weekly
- Check Search Console for new coverage issues.
- Review indexed pages for bad canonicals or duplicate URLs.
- Look at blog clicks and impressions.

## Monthly
- Re-test Core Web Vitals on Home, /estilos, /galeria, and /precios.
- Check if FAQ answers still render in visible DOM.
- Review whether `/seguimiento` and `/order` are behaving as intended.
- Post at least one authority signal piece: YouTube, Reddit, or a community mention.

## Quarterly
- Re-run backlink discovery once search engines have more graph data.
- Review whether any "convert photo into X" search terms justify a free-preview page.
- Check if the blog needs one new post or a major refresh of an old one.

## Watch For
- Canonical drift back to homepage.
- Hidden FAQ content.
- New duplicate schema.
- Slow LCP on mobile.
- Any sign that users want a preview before ordering.

## Decision Rule
- Build a free-preview entry point only if tool-intent queries start getting impressions and users bounce from the paid flow.
- Do not build it just because competitors have one.
